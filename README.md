# laudo-gi

Máscara de laudo GI endoscópico/colonoscópico pediátrico — app single-file (HTML/CSS/JS, offline-first).

- Uso: abra `index.html` (ou acesse via GitHub Pages).
- Dados salvos localmente (localStorage); backup/restauração JSON embutidos.
- Sync opcional entre PCs via Google Drive (appDataFolder) — requer colar um OAuth Client ID nas Configurações do app.
- Nenhum dado pessoal é enviado a este repositório ou host; apenas a biblioteca de diagnósticos/critérios.
